선언 헤더 :	unistd, stdlib, string

선언 상수 :	STDIN(0), STDOUT(1), STDERR(2)
		  	T_END(3), T_PIPE(4), T_BREAK(5)
			에러 메세지들

선언 구조체의 내용
 - int fd[2];	// 파이프 오픈용 파일 디스크럽터
 - int size;	// 명령어 문자열의 크기
 - int type;	// 명령어 뒤에 나오는 타입(end, pipe, break)
 - char **argv;	// 명령어 문자열 모음
 - ptr *prev;	// 이전 구조체 주소
 - ptr *next;	// 다음 구조체 주소


선언 함수
 - 문자열 관리 함수
 - 에러 출력 함수
 - 파싱 함수
	- 파이프와 엔드의 위치 찾기
	- 찾은 위치를 바탕으로 명령어 구조체 채우기 만들기
 - 실행 함수
	- 명령어의 첫 번째 인자가 cd이면 cd 예외처리하기
		- 명령어 길이가 2보다 작으면 인자 에러
		- chdir함수로 명령어에 적힌 디렉토리를 찾을 수 없으면 경로 에러
	- 명령어의 타입을 확인하고 파이프이면 pipe함수를 이용해 파이프 열기, 실패시 에러 출력 후 종료
	- fork함수를 사용해 자식 프로세스를 생성, 실패시 에러 출력 후 종료
	- 자식 프로세스일 때
		- 지금 명령문이 파이프인데 입력 디스크럽터(fd[1])이 닫혀있는 경후 에러 출력 후 종료
		- 이전 명령문이 파이프인데 출력 디스크럽터(fd[0])이 닫혀있는 경후 에러 출력 후 종료
		- 명령문을 실행하고 만일 잘못되면 에러 출력 후 종료
	- 부모 프로세스일 때
		- waitpid함수로 자식 프로세스 기다리기
		- 파이프가 열려있는 상태이면 현재 명령문의 입력 디스크럽터 닫기. 만일 다음 명령문이 없거나 파이프가 아니면 출력 디스크럽터 닫기.
		- 만일 이전 명령문이 파이프였으면 이전 명령문의 출력 디스크럽터 닫기.
